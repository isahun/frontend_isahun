# Frontend Isahun

This project has three iterations, each on its own branch:

- **feature/vanilla-implementation** – plain HTML, CSS, and JS  
- **feature/sass-implementation** – styled with Sass  
- **feature/tailwind-implementation** – Tailwind CSS with Vite  

## Installation

Clone the repo and install dependencies:

git clone https://github.com/isahun/frontend_isahun.git
cd frontend_isahun
npm install
Switch to the branch you want:

git checkout <branch-name>
Running
Vanilla JS – open HTML files in your browser

Sass – compile with npx sass src/styles:dist/styles

Tailwind (Vite) – start dev server: npm run dev or build: npm run build
